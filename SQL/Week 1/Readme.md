Week 1 키워드

```
SQL 문제를 풀때 역순으로 키워드를 잡아 풀이를 진행하면 좋다.
SELECT 는 출력에 해당하고 FROM은 가져올 구간으로 들어가고
나머지는 상황에 맞게 조건을 넣어준다.


DISTINCT : 중복을 제거한다.
WHERE : 특정한 조건의 집합을 가져온다.
first_name / last_name으로 구성된 컬럼을 (성, 이름) 형태로 변경하기 위해서는 '|| ||' 형태를 사용한다.

mod(x, y) : x를 y로 나눈 후 나머지값을 산출하는 함수다.
char_length(x) : 글자 수를 세는 함수
upper : 대문자 변환 함수
lower : 소문자 변환 함수
```

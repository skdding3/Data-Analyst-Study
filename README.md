### 그로스팀 문화

- 책 읽기 진행시 모든 내용을 다 넣는다는 생각보다는 중요한 내용을 위주로 정리한다.

- 통계적인 실험도 진행하도록 하여, 프로젝트를 하나 만들수 있도록 진행한다.
